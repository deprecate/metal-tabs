Tabs

1. Pills style tabs with some tabs disabled. Style is given via the predefined type attributes.

2. Pills style tabs completely disabled. Style is given via the predefined type attributes.

3. Regular style tabs with some tabs disabled. Style is given via the elementClasses attribute.

4. Tabs completely disabled with no style given. If no style is given it falls back to the regular one.

5. Custom style tabs with some tabs disabled. Style is given via the elementClasses attribute, type attribute set to none.
